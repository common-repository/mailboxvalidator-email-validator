Email verification for WordPress forms. Block disposable, block spam, block invalid email, block free email and role-based email.

== Description ==

This email validation plugin helps you to block disposable email, free email, invalid email or role-based email from using your services or input form.

It helps to block spam, temporary emails and invalid email address messing around your site. You don’t have to master the hard part of adding shortcode into page source to use this plugin, but to activate, configure and you’re ready to prevent all spam signups from invalid email addresses.

You can [sign up for a free API key](https://www.mailboxvalidator.com/plans#api "MailboxValidator API key sign up") at mailboxvalidator.com, that will allow you to validate up to 300 emails every month for free!

== Installation ==

### Installation using WordPress dashboard

1. Select **Plugins -> Add New**.
1. Search for "MailboxValidator Email Validator".
1. Click on *Install Now* to install the plugin.
1. Click on *Activate* button to activate the plugin.
1. Get your [MailboxValidator API key](https://www.mailboxvalidator.com/plans#api "MailboxValidator API key sign up") before start using the plugin.
1. Configure the plugin settings, including your API key, domain blacklist and switch on or off the valid, disposable and free and role-based email validator, and design custom error messages.

### Manual Installation

1. Upload the plugin files to the `/wp-content/plugins/mailboxvalidatoremail-validator` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Get your MailboxValidator API key through [here](https://www.mailboxvalidator.com/plans#api "MailboxValidator API key sign up").
1. Configure the plugin settings, including your API key, domain blacklist and switch on or off the valid, disposable, free and role-based email validator, and design custom error messages.

== Frequently Asked Questions ==

= How can this plugin help to validate form details?  =

This plugin will validate those email and make sure that it is valid or not belong to disposable, free email or role-based. In other words, it will reject those email which is invalid, disposable or free email, depends on your settings.

= What is role-based email address?  =

A role-based email address is the email address that front part like admin@, sales@, support@ and so on. 

= Do I need an API key to use this plugin? =

Yes. You can sign up for a FREE API key from [MailboxValidator](https://www.mailboxvalidator.com/plans#api "MailboxValidator API key sign up") to use this plugin.
